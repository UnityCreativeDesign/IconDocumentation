String

The full path to where the sample will be imported, under the project assets folder.

It is in the form of Project Full Path/Assets/Samples/Package Display Name/Package Version/Sample Display Name/.

If the sample is an asset package (a .unitypackage file), this value won't be taken into consideration during import
