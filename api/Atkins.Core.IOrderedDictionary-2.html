    TValue value

    The value of the entry to add.
    Returns
    Type Description
    Int32

    The index of the newly added entry
    Remarks

    You can also use the System.Collections.Generic.IDictionary{TKey,TValue}.Item(TKey) property to add new elements by setting the value of a key that does not exist in the IOrderedDictionary<TKey, TValue> collection; however, if the specified key already exists in the IOrderedDictionary<TKey, TValue>, setting the Item(TKey) property overwrites the old value. In contrast, the Add method does not modify existing elements.
